User remarque sur instance C (sandbox migration) que les images s'affichent visellement sur le site mais n'existent PAS dans la médiathèque WordPress. **Hypothèse** : images servies via redirection (CDN, stockage externe) plutôt que fichiers physiques en `/wp-content/uploads/`. **À vérifier** : inspect URL requête image (DevTools), vérifier présence/structure de `wp-content/uploads/`, vérifier config plugin optimisation images (WPO, Smush, CloudFlare, etc.). **Risque** : redirections cassent lors d'une migration définitive ou changement de serveur media.
