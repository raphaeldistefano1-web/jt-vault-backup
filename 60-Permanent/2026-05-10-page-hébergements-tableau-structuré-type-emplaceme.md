User demande tableau structuré pour pages hébergements/bungalows. Layout cible : **Col 1** (gauche) = type d'hébergement (ex: « Bungalow 2 pers. »), **Col 2** = emplacement/localité, **Col 3+** = prix par saison (1 colonne = 1 saison, 1 cellule = 1 prix). Exemple : `Bungalow XL | Plage Nord | 180€ (creuse) | 240€ (moyenne) | 320€ (haute)`. Fichiers : `page-hebergements.php`, `page-bungalows.php` (instances A + C). Coordonner avec page-restauration.php si tarifs présents.
